# SUNTA: 살타 할아버지
> 호주에서의 건강하고 안전한 야외 활동을 위해 탄생한 위치기반 UV 지수 제공 서비스입니다. 
본 서비스는 세계 최고 수준의 자외선 환경인 호주에서, 평소 다양한 액티비티를 즐기는 모든 연령대를 포함하되, 특히 피부암에 가장 취약한 40~60대 액티비티 참여자를 핵심 타깃으로 설정합니다.
실시간 UV 지수와 맞춤 선크림 추천 및 시간별 체크리스트를 통해 꼼꼼한 자외선 차단을 생활화하도록 돕습니다. 
특히, 여름에 서핑이나 액티비티를 즐기는 ‘산타할아버지' 캐릭터가 UV 지수와 선크림 도포 여부에 따라 변화하고, 사용자들에게 친근하고 시각적인 경고를 제공하는 차별화된 자외선 관리 앱입니다.

|------------|----------------------------------------------|
| `Feat`     | 새로운 UI 및 기능 구현                       |
| `Add`      | 부수적인 코드·폰트·에셋·주석 등 추가         |
| `Fix`      | 버그 및 오류 해결                            |
| `Refactor` | 리팩토링                                     |
